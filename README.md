### Multi-scale convolutional neural networks for cloud segmentation

### Requirements
- Python 3
- Tensorflow 2.0
### datasets

- [38-Cloud dataset](https://www.kaggle.com/sorour/38cloud-cloud-segmentation-in-satellite-images)
